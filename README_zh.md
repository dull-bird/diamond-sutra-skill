# 金刚经 · Skill

> 「凡所有相，皆是虚妄。若见诸相非相，即见如来。」

[English](README.md) | **中文** | [主页](https://dull-bird.github.io/diamond-sutra-skill/)

这是一个开源 **AI Agent Skill**：把《金刚经》的般若智慧整理成 AI 可以遵循的义理体系和说话方式。它不装神弄鬼，不算命，不扮演高高在上的大师，只在你焦虑、悔过去、忧未来、被工作压力或身份标签压住时，把问题带回日常生活里讲清楚。

本 Skill 主要从三层材料蒸馏而成：

- 南怀瑾 **《金刚经说什么》**：负责义理脉络和经文讲解。
- 费勇 **《金刚经修心课：不焦虑的活法》**：负责现代焦虑、身份和日常烦恼的解释方式。
- 鸠摩罗什译《金刚经》公版经文，以及 [`scripture-en.md`](scripture-en.md) 中整理的英文译法和术语对照。

## 安装

推荐使用：

```bash
npx skills add dull-bird/diamond-sutra-skill
```

也可以手动 clone 到 Claude Code 的 skills 目录：

```bash
git clone https://github.com/dull-bird/diamond-sutra-skill ~/.claude/skills/diamond-sutra
```

安装后可直接唤醒：

```text
/diamond-sutra
```

## 用法

有两种唤醒方式。

**1. 显式唤醒**

直接输入 `/diamond-sutra`，然后请教佛法问题，或描述你正在经历的烦恼。

**2. 自动触发**

当你自然提到焦虑、职场压力、身份标签、放不下过去、担心未来时，支持 Agent Skills 的工具可以根据描述自动调用它。

示例：

> **你：** 最近项目老被打回，我觉得自己特别失败，脑子一直停在昨天。
>
> **Diamond Sutra Skill：** 《金刚经》说「过去心不可得，现在心不可得，未来心不可得。」昨天那股水已经流走了。你的难受是真的，但你不必把昨天的难受做成今天的身份。

## 设计

这个 Skill 有两层。

**义理层：** 以《金刚经》的般若思想为底座，围绕破四相、无相布施、无法可得，以及「应无所住，而生其心」展开。

**说法层：** 语言温和、通俗、接地气，不居高临下，不神秘化，不宣称证悟；尽量用现代生活里的例子讲古老智慧。

## 作为普通 Prompt 使用

如果你的 AI 工具不支持 Agent Skills，也可以直接复制 `SKILL.md` 的全部内容，粘贴到任意聊天模型里，然后说：

```text
请你严格遵循这份 SKILL.md 定义的义理体系与角色设定，和我展开对话。
```
