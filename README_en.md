# 💎 Diamond Sutra AI Skill

> *"All phenomena are illusions. If you see that all phenomena are not phenomena, you see the Tathagata."*

[English](README_en.md) | [中文](README.md)


This is an AI Skill (a large language model persona plugin) distilled primarily from the classical commentaries of Master Nan Huai-Jin and modern interpretations by Professor Fei Yong.
It can instantly transform any tool that supports AgentSkills (like Claude Code, Gemini CLI, etc.) into an enlightened, world-wise, and wonderfully humorous Dharma master, ready to help you untangle your work and life anxieties anytime, anywhere.

## 📦 One-Click Installation

**Method 1: Using npx skills (Recommended)**

If your environment supports the `npx skills` command, you can run the following to install directly:

```bash
npx skills add dull-bird/diamond-sutra-skill
```

**Method 2: Manual Git Clone (Install for Claude Code, etc.)**

If your terminal supports AgentSkills and is configured to read `~/.claude/skills/`:

```bash
git clone https://github.com/dull-bird/diamond-sutra-skill ~/.claude/skills/diamond-sutra
```

—
Whichever method you choose, after installation, you can instantly invoke it in your terminal interaction interface:
```
/diamond-sutra
```

## 💬 Start Experiencing

There are two ways to invoke this skill:

**Method 1: Explicit Invocation**  
Simply type `/diamond-sutra` to start pouring out your troubles to "him" or asking profound Buddhist questions.

**Method 2: Smart Auto-Invoke (Passive Trigger)**  
Because this skill has built-in situational triggers in its description, you don't even need to type the command! Just complain naturally in your chat:
- ❓ *"My project keeps getting rejected lately, work is super stressful, and I feel like the labels society forces on me are suffocating me. How would Buddhism view this?"*

The LLM agent will automatically detect your "anxiety", "work pressure", and "identity confusion", and seamlessly invoke this "Digital Dharma Master" in the background to counsel you.

## 🪷 Distinctive Features of this Skill

This skill is distilled using a true Two-Layer Architecture:

1. **Doctrine Layer**: Incorporates the supreme Prajna wisdom of the Diamond Sutra, ranging from "breaking the four forms" to "skillfully protecting your thoughts", strictly based on the original scripture and renowned commentaries. No superstitions, and it never loses touch with the secular world.
2. **Persona Layer**: No mystical posturing, and never condescending. The language is highly colloquial and amiable. He frequently quotes well-known poetry and excels at using the most modern metaphors (like "camera" and "social labels") to explain the oldest wisdom. Always sticking to: "Popular and humorous, extensively quoting, combined with life, and absolutely non-preachy."

## 🧩 Direct Prompt Application (General Usage)

If you aren't accustomed to terminal AI tools, you can simply copy the entire contents of `SKILL.md` in the root of this project and paste it into any web-based large language model (ChatGPT, Claude 3, Kimi, etc.), telling it: **"Please strictly follow the doctrine system and persona settings defined in this SKILL.md for our conversation."**

Then, you can experience a hearty long talk as if you were receiving face-to-face guidance from an eminent monk.
